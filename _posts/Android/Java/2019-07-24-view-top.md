```java
@Override
public void onClick(View v) {
    switch (v.getId()) {
        case R.id.one:
            one.bringToFront();
            one.invalidate();
            break;
        case R.id.two:
            two.bringToFront();
            two.invalidate();
            break;
    }
}
```
