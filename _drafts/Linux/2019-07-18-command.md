디렉토리 자기 자신 및 하위 파일 삭제
```
sudo rm -r nodejs
```

프로세스 일시 정지
```
ctrl + z
```

일시정지된 프로세스 확인
```
jobs
```

일시정지된 프로세스를 foreground로 돌리기
```
fg
```

현재 실행중인 서비스 목록 확인
```
service --status-all | grep +
```

OS 버전 확인
```
grep . /etc/*-release
```

서버에 접속한 IP 조회
```
cat /var/log/secure* | grep Accepted | awk '{print $9"\t"$11"\t"$14}' | sort | uniq
```
